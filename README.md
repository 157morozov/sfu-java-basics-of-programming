# 🏫 Обучение Java от СФУ

> Репозиторий с прохождением курса начального обучения языку программирования Java

---

## 🗂️ Структура проекта

```
project/
├── src/
│   └── Main.java       # Главное меню — точка входа всего проекта
├── pr1/
│   └── Main.java       # Задание 1
├── pr2/
│   └── Main.java       # Задание 2
└── ...
```

### `src/Main.java` — главный файл

Точка входа всего проекта. Запускает бесконечный цикл с меню, в котором можно выбрать номер задания или выйти из программы.

```
=== Учебный проект по Java ===

Выберите задание:
  1. Сортировка матрицы по сумме модулей элементов строк
  q. Выйти

Ваш выбор: _
```

### `pr1/`, `pr2/`, ... — папки с заданиями

Каждая папка соответствует отдельному заданию курса и содержит файл `Main.java` с его реализацией. После завершения задания управление автоматически возвращается в главное меню.

---

## 🚀 Запуск

Запустить `src/Main.java` как основной класс проекта.
